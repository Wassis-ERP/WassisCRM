# WassisCRM Workspace

Workspace do frontend CRM da W.Assis. A aplicacao principal esta em `nexus-crm`, e os scripts da raiz encaminham os comandos para essa pasta.

## Comandos

```powershell
npm install
npm run dev
npm run build
npm run lint
```

Veja a documentacao completa em `nexus-crm/README.md`.
